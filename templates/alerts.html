How to set up your Discord webhook

  1. Open Discord and go to the channel where you want alerts (or create a new one like #trading-alerts).
  2. Click the ⚙️ Edit Channel gear → Integrations → Webhooks → New Webhook.
  3. Give it a name like Quant Bot, then click Copy Webhook URL.
  4. Paste it in the field below and click Test Connection to verify it works.
